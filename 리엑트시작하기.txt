리엑트 시작하기

1. npm 모듈 초기화
    npm init

2. npx create-react-app 프로젝트명
    npx : npm과 달리 해당하는 모듈이 없으면, 설치후 실행한다.
        create-react-app 이 없으면 설치후 실행한다.
    CRA : 노드 기반의 리액트 프로젝트 탬플릿을 만들어주는 모듈.
    npx create-react-app first-app

3. 프로젝트 시작
    cd first-app
    npm start

4. http://localhost:3000 에서 실행중...
    노드 서버는 3000번에서 3030번으로 바꿔서 실행해야 됨.(port충돌)
